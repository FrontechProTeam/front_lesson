# homework5
## hello_array
arrayの中身が一つずつ出力されるようにしましょう
```
入力: const array = ["h","e","l","l","o"]
出力:
h
e
l
l
o
```

## sum_array
arrayの中身の数字を全て合計して出力しましょう
```
入力: const array = [1,2,3,4,5]
出力: 合計は15です！
```

## hello_array2
文字列を一文字ずつ配列に入れてみましょう
```
入力: const hello = "hello"
出力: [ 'h', 'e', 'l', 'l', 'o' ]
```

## lucky7
0~9のランダムな文字を生成し、7だった時は「ラッキ－７😎」と出力しましょう
```
出力例1: 3
出力例2: ラッキー７😎
```
